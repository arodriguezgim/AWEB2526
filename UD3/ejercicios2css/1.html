Ejercicio 1: Modelo de Cajas Básico

Esta es una caja con contenido.

Observa cómo el padding, margin y border afectan su tamaño total.

📏 Información del Modelo de Cajas:

Ancho total = width + padding-left + padding-right + border-left + border-right

En este caso: 300px + 20px + 20px + 5px + 5px = 350px

Alto total = height + padding-top + padding-bottom + border-top + border-bottom

En este caso: 200px + 20px + 20px + 5px + 5px = 250px

Espacio ocupado (con margin): 350px + 30px + 30px = 410px de ancho
