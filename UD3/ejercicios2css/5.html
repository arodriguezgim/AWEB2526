Ejercicio 5: Centrar Elementos con Margin Auto

💡 Técnica de centrado con margin:

margin: 0 auto → Centra horizontalmente (0 arriba/abajo, auto izq/der)

margin-left: auto → Empuja el elemento a la derecha

margin-right: auto → Empuja el elemento a la izquierda

Nota: Solo funciona con elementos de bloque que tengan un ancho definido

Alineada a la izquierda

margin-left: 0

margin-right: auto

Centrada

margin: 20px auto

Alineada a la derecha

margin-left: auto

margin-right: 0

✅ Resultado esperado:

✓ Contenedor principal centrado en la página

✓ Caja roja alineada a la izquierda

✓ Caja azul centrada perfectamente

✓ Caja verde alineada a la derecha
